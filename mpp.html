MyPartPros

Improve warehouse efficiency and track worker productivity.

Problem

The founders of MyPartPros, two brothers
working in a car dealership parts department,
identified a critical inefficiency in the parts-picking
process. Many parts departments still rely on
outdated systems, such as pen and paper,
which leads to excessive time spent on delivering
parts to customers. This inefficiency negatively
impacts both productivity and customer satisfaction.

Solution

To address this issue, a mobile application
was developed to connect seamlessly with
CDK Global and auto parts departments.
The app allows users to optimize parts picking
routes, starting with the closest items and
progressing to the furthest. This streamlined
approach significantly reduces the time
required to complete deliveries compared to
traditional methods.

My Role

I was responsible for creating a cohesive
style guide, conducting research, and designing
wireframes for both the admin portal and
the Android application.

Scope

The design process for this project spanned
six months, from initial concept development
to final destination.

Web App

View on Figma!

Warehouse management

On the Warehouse Management screen, an admin user can select a specific dealership they are associated with and view details for all orders, including unpicked items and those in staging. This screen provides critical information such as part numbers, quantities, descriptions, invoice numbers, statuses, and more to streamline operations.

Unpicked parts

The Unpicked Parts section displays all items that have not yet been picked, along with detailed information as described above.

Staged

In the Staged section, the user can select a Dealer Management System (DMS) and review both completed and incomplete orders for further processing.

Admin section

In the Admin Section, an admin user has tools to manage employee accounts. They can view a list of employees by their roles, reset passwords, edit account details, and create new user accounts to ensure efficient management of system access.

Mobile App

View on Figma!

These screens allow users to select the dealership they are working at and choose from various tasks such as picking parts, starting a route, or retrieving an automotive part. The application provides feedback by flashing colors on the screen to indicate whether the correct or incorrect part has been selected.

In the Staging section, users can select a Dealer Management System (DMS) account, review specific orders, and print a designated number of labels for each automotive part to streamline the organization process.
