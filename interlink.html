Interlink AI

Simplifying the process of delivering the correct prescriptions to the right patients.

Problem

Interlink AI approached Beeso Studio to
address issues with their product,
scripClip, and to provide a UI reskin and
update to improve its usability and aesthetics.

Solution

The solution involved moving away from
the outdated, traditional pharmacy UI
design and transitioning to a modern,
clean, and user-friendly interface.

My Role

I was responsible for redesigning the UI
and UX of scripClip. I proposed moving
away from the conventional pharmacy
design style and embracing a more
contemporary and polished look to
better align with modern user expectations.

Scope

My work on this project spanned 4–5
months, during which I contributed to
the redesign and improvement of several
key features and workflows.

Software

View on Figma!

Patient Search

This updated screen allows pharmacy users to search for a patient, select their profile, and retrieve their prescription. The application
also enables users to locate prescriptions using sound notifications
or through the express pickup process for faster service.

Express Pickup

This feature simplifies the prescription retrieval process by allowing pharmacy users to scan a driver’s license for quick access to prescriptions. It functions similarly to the patient search feature but is optimized for speed and efficiency.

Packaging

Pharmacy users can scan one or multiple prescriptions and package them into either a new scripClip bag or an existing one. This process streamlines packaging while maintaining organization.

Packaging Match

This screen enables users to match a prescription vial with a prescription sent by a doctor. Once a match is confirmed, the
user can proceed to package the prescription efficiently.

Return to Stock

Pharmacy users can return unclaimed prescription vials back to inventory
after a specified period. This feature ensures better stock management and minimizes wasted resources.

scripClip Details

This functionality allows users to search for a patient and access detailed information, including when a prescription was packaged and picked up.

Clear Patient Prescription Record

This feature is designed to remove a scripClip record from the database
when the bag is no longer found, allowing pharmacy staff to restart the packaging process as needed.

Alerts

The Alerts screen provides an overview of all issues within the pharmacy.
Users can view and address specific alerts or dismiss them after resolution.

Workstation Settings

This customizable settings page lets users adjust their workstation
preferences, including default login screens, flash colors, patterns,
durations, and sound settings. These settings ensure pharmacy users
can easily identify and retrieve the correct scripClip bag while
minimizing errors.
